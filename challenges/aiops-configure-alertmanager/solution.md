# Solution: Configure Alertmanager Routing

## What the validator checks

- <value> not found.
- Invalid alertmanager config. Run: amtool check-config <value>

## Solution

Address each check above in order:

**<value> not found.**
  → 1. Check config exists

**Invalid alertmanager config. Run: amtool check-config <value>**
  → 2. Validate config syntax
